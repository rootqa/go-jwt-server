# Go Example JWT Server
Go-based JWT Tokenizer Implementation
Algo: "HS256"

## Handler

```shell
http://127.0.0.1:8080
```

## JWT Token Server

```shell
http://127.0.0.1:8081/token
```

## Variables

- SigningKey (256-bit JWT secret)

## Variables
Header =>	claims["authorized"]

Payload =>	claims["wallet"]

Token Time =>	claims["expired"]
